TubeMateClone
This program is a youtube downloader

Dependencies:
- pytube or pytubefix (this one worked better than pytube, although they work the same)
- FFmpeg (this must be installed on your device for the program to work)

How to use:
- Get the link of the youtube video you wish to download in the formats:
  - https://youtu.be/id
  - https://www.youtube.com/watch?v=id
- Paste the link when prompted
- enter 'y' at the confirmation prompt if the title, length and channel is what you're looking for. enter 'n' otherwise, and try a different link
- when prompted for resolution, enter the code/number for the corresponding resolution 
- video will begin downloading and running background processes.
- when everything is complete, your video should be found in a directory called "out" in the same location as where you run the script from
